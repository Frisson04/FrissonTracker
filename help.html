Guide d'utilisation complet

Traceur Automatique

Découvrez un outil de tracking intuitif et parfaitement adapté à vos besoins, avec une configuration simplifiée pour une prise en main immédiate.

Exploration de l'Interface

Panneau de Configuration

Accédez à un tableau de bord interactif vous permettant de configurer votre tracker avec une précision inégalée, le tout dans un environnement visuel et ergonomique.

Mode Personnalisation

Libérez votre créativité avec notre système de personnalisation avancé : redimensionnez, repositionnez et ajustez chaque élément selon vos préférences pour créer une interface véritablement unique. Maintenez la touche Shift enfoncée pour activer le Smart Guide - un outil d'alignement intelligent qui vous aide à positionner parfaitement vos objets et catégories.

Mode Streaming (OBS)

Intégrez facilement votre tracker comme overlay dans OBS avec notre configuration optimisée pour le streaming. Une simple capture de fenêtre suffit pour afficher votre progression en temps réel à votre audience.

Tips : Utilisez le mode "Capture de fenêtre" dans OBS et sélectionnez l'application. Activez la transparence dans les filtres d'obs pour un overlay parfaitement intégré à votre scène.

Guide Complet des Fonctionnalités

Fonctions de Base

Personnalisez l'ambiance visuelle de votre tracker en modifiant l'image d'arrière-plan selon vos goûts.

La connexion à votre compte est essentielle pour débloquer toutes les fonctionnalités de personnalisation de votre overlay.

Créez et organisez vos catégories avec une grande flexibilité : ajoutez des codes couleur personnalisés et ajustez la taille des icônes pour une expérience visuelle optimale.

Une fois connecté, configurez facilement vos Items et Locations grâce à notre tableau interactif. Nous explorerons les paramètres avancés dans les sections suivantes.

Exportez et importez vos configurations de packs en quelques clics. Pensez à sauvegarder régulièrement votre travail pour préserver vos personnalisations !

Paramètres des object reçu

Associe automatiquement le nom de l'image avec l'objet correspondant dans Archipelago.

Sélectionnez librement la catégorie à associer à chaque objet pour une organisation parfaitement adaptée.

Importez vos propres images ou utilisez des liens externes pour personnaliser complètement l'apparence de vos objets.

Mode Compteur (Counter)

Ce mode affiche dynamiquement la quantité d'un même objet possédé. Idéal pour suivre les objets accumulables comme les étoiles ou les pièces.

Mode Images Différentes (Different images)

Permet d'afficher des images distinctes en fonction du nombre d'exemplaires possédés. Parfait pour représenter les évolutions d'objets (comme le grappin dans Zelda) ou les améliorations progressives.

Exemple concret : Le grappin évolue visuellement d'une version courte à une version longue selon votre progression.

Configuration des Locations

Image de Checkmark

Sélectionnez l'image qui s'affichera comme indicateur de validation lorsque la location sera complétée. Cette fonctionnalité vous permet de personnaliser visuellement la confirmation d'achèvement.

Exemple : Vous avez Mario comme objet obtenu, vous pouvez afficher un checkmark sur son icône dès que vous accomplissez la location associée par exemple "Win with Mario".

Gestion de l'Ordre d'Affichage

Déterminez précisément l'ordre de priorité d'affichage des images pour représenter visuellement votre progression.

Cas d'utilisation : Pour les trophées de coupe dans Mario Kart, vous pouvez configurer l'affichage séquentiel des images (Bronze → Argent → Or).
